## 소개

react-native, Spring Boot 기반 애플리케이션으로,
바코드 API, FCM 알람 기능 등을 통해 유통기한 관리를 도와주며,
YOLOv5 객체 인식 알고리즘을 기반으로 레시피를 쉽게 검색 할 수 있고,
최근 조회 레시피와 북마크 기록을 통해 사용자에게 적절한 레시피를 추천합니다.

## 개발 환경

- Window 11
- MacOS Monterey

## 사용 기술

- react-native
- Spring Boot
- MySQL @8.0.28
- Flask & Yolov5
